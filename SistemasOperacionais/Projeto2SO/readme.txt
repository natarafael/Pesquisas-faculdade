Pré-requisitos:
SO windows (foi usado a ultima versão do win10 para implementação)

Compilador: JAVA (IDE usada: intellij da Jetbrains)

Configurações necessárias e instruções:
Deve-se alterar o diretorio do relatório de saida e da pasta backup, respectivamente nas linhas 37 e 50 do arquivo TDisco.java
obs: pasta backup deve ser criada no computador com antecedencia e deve estar vazia, o arquivo saida, se existente deve ser excluido antes da execução.
Deve-se também mudar os diretorios dos arquivos dentro do backup, para o correspondente em sua maquina, localizados nas linhas 61 e 66 do arquivo TDisco que representam inclusão e exclusao dos arquivos, respectivamente.
obs: o diretorio deve permanecer no formato
"[seu diretorio]\\backup\\arquivo"+i+".txt"
para funcionamento correto do simulador.

Foi adicionado um arquivo saida.txt como exemplo para visualizar processo de execução, que deve ser excluido antes de executar o programa
